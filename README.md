# 제주 가이드 🏝️

제주도 여행을 위한 추천 장소 정보를 제공하는 웹 애플리케이션입니다. 7개월간의 제주도 장기여행과 5년간의 거주 경험을 바탕으로 엄선한 맛집, 카페, 풍경 명소 등을 소개합니다.

## 🌟 주요 기능

### 1. 장소 탐색
- **지도 보기**: 제주도 전체 지도에서 추천 장소를 마커로 확인
- **목록 보기**: 카드 또는 테이블 형식으로 장소 목록 확인
- **상세 정보**: 각 장소의 사진, 설명, 주소, 운영시간 등 상세 정보 제공

### 2. 검색 및 필터
- 장소명, 주소, 설명으로 통합 검색
- 카테고리별 필터링 (맛집, 카페, 풍경, 술집, 숙소 등)
- 페이지당 표시 개수 조절

### 3. 나만의 장소
- 마음에 드는 장소를 저장하여 개인 컬렉션 생성
- 로컬 스토리지를 활용한 데이터 저장

### 4. 여행 팁
- 제주도 대중교통 정보
- 뚜벅이 여행자를 위한 꿀팁
- 계절별 추천 정보

### 5. 관리자 기능
- **장소 추가**: 새로운 추천 장소 등록
  - 장소명, 카테고리, 주소, 상세 설명 입력
  - 다중 이미지 업로드 지원
  - 운영시간, 연락처 등 부가 정보 입력
- **장소 수정**: 기존 장소 정보 업데이트
  - 모든 장소 정보 수정 가능
  - 이미지 추가/삭제 기능
  - 실시간 미리보기 지원
- **장소 삭제**: 더 이상 운영하지 않는 장소 제거
- **인증 시스템**: 관리자 전용 로그인

## 🔐 관리자 사용 가이드

### 관리자 페이지 접속
1. URL 직접 입력: `/admin`
2. 관리자 계정으로 로그인

### 장소 추가하기
1. 관리자 페이지에서 "새 장소 추가" 버튼 클릭
2. 장소 정보 입력:
   - 장소명 (필수)
   - 카테고리 선택 (맛집, 카페, 풍경 등)
   - 주소 입력
   - 상세 설명
   - 이미지 업로드 (드래그 앤 드롭 지원)
3. "등록하기" 버튼 클릭

### 장소 수정하기
1. 관리자 페이지의 장소 목록에서 수정할 장소 선택
2. "수정" 버튼 클릭
3. 정보 수정 후 "저장" 버튼 클릭

### 장소 삭제하기
1. 관리자 페이지의 장소 목록에서 삭제할 장소 선택
2. "삭제" 버튼 클릭
3. 확인 대화상자에서 "확인" 클릭

## 🚀 시작하기

### 필수 요구사항
- Node.js 16.0.0 이상
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/yourusername/jeju-guide.git
cd jeju-guide

# 의존성 설치
npm install
# 또는
yarn install

# 개발 서버 실행
npm start
# 또는
yarn start
```

개발 서버는 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.

### 빌드

```bash
npm run build
# 또는
yarn build
```

빌드된 파일은 `build` 폴더에 생성됩니다.
